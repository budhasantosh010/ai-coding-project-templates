# runs/ — evidence from real executions

When you actually RUN something (a test suite, the pipeline, a build) and want to keep the
proof, drop it here. One file per run:

```
YYYY-MM-DD-what-was-run.md   (or .txt / .log)
```

This is where "evidence level E3/E4/E5" claims get their backing. If `CURRENT_STATE.md` says
something is verified working, the run that proved it should live here so anyone can check.
