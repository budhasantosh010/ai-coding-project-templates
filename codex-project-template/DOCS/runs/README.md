# Run reports

Create one immutable report per meaningful execution:

`YYYY-MM-DD-HHMM-<run-id>.md`

Record:

- input identifiers and hashes;
- code/config version or snapshot;
- command/profile;
- start/end time;
- stage durations;
- result;
- warnings/failures;
- output paths;
- cost;
- verification evidence.
